Disease Risk Prediction using Machine Learning

Objective:
Predict disease risk levels using patient health indicators.

Algorithms:
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

Results:
- KNN Accuracy: 80.4%
- Naive Bayes Accuracy: 74.4%

Technologies:
Python, Pandas, NumPy, Matplotlib, Scikit-learn
